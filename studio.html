Subtitles Troubleshooting

If automatic subtitles fail, ensure you are using a modern browser (Chrome/Edge recommended) and have granted microphone permissions.

The application attempts to use Vosk (offline-capable) first, then falls back to the browser's built-in Web Speech API.
