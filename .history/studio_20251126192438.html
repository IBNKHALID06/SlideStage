Keyboard: Left/Right arrows to change slides. Drag webcam box to reposition. Resize from corner. Use the "Record This Tab" button for capture.

Record This Tab

To record slides, webcam, subtitles, cursor, and mic audio, use your browser's built-in tab capture:

  1. Open Chrome or Edge.
  2. Click the browser's "Share" icon or use extensions to capture this tab.
  3. Pick "Record This Tab"; include microphone audio.
  4. Stop recording to save a .webm file locally.

Subtitles (Vosk WASM)

Real subtitles require an offline Vosk model. For privacy and offline use, place a small model in assets/models/en/. See instructions on the homepage or README.

  1. Download a small English model (e.g., vosk-model-small-en-us-0.15).
  2. Unzip it into assets/models/en/ so that model.json is inside that folder.
  3. Reload the page and click "Enable Subtitles".
